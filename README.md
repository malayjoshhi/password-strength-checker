# 🔑 Password Strength & Data Breach Checker

An advanced Python tool to evaluate password entropy, check character diversity, detect weak dictionary patterns, and verify if credentials have leaked in known data breaches using the **HaveIBeenPwned API** with **k-Anonymity privacy**.

---

## 🌟 Features
Information entropy scoring and character diversity analysis.
Common dictionary pattern and sequence matching.
Fully anonymous password leak checking without sending credentials online.

---

## 🚀 Quick Start

```
# Analyze interactively with secure masked typing
python main.py

# Analyze specific password and check HaveIBeenPwned breaches
python main.py -p "MyP@ssw0rd2026!" --check-breaches
```
